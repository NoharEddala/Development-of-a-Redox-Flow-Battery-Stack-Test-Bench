# 🔋 Development of a Redox Flow Battery Stack Test Bench

**Goal:**  
To design and develop a laboratory test bench for the characterization, monitoring, and performance evaluation of experimental redox flow battery stacks.

---

## 🔧 Overview

This project was developed during my work as a **Work Student at Institut für Regenerative Energie Systeme (Hochschule Stralsund)**.

The test bench was designed to support experimental investigations of redox flow battery stacks by providing controlled operating conditions, automated measurements, and real-time monitoring of key electrical parameters.

The setup integrates:

- Battery stack testing under CC and CV operating modes
- Real-time voltage and current measurement
- Data acquisition and monitoring system
- Automated performance evaluation and validation

---

## 🧩 System Architecture

- **Test Bench:** Custom-built experimental setup for battery stack characterization
- **Measurement System:** Voltage and current sensing for performance monitoring
- **Control System:** Automated operation under Constant Current (CC) and Constant Voltage (CV) modes
- **Data Acquisition:** Continuous recording of electrical parameters for analysis and validation

---

## ⚙️ Key Features

- Automated battery stack testing and monitoring
- Operation in both **CC (Constant Current)** and **CV (Constant Voltage)** modes
- Real-time acquisition of voltage and current data
- Performance characterization and system validation
- Modular architecture for different stack configurations
